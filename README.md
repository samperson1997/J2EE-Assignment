# J2EE_homework
J2EE homework

## Requirements
* 客户登录, 跟据客户的ID／PW, 查询订单情况, 并根据具体情况, 返回不同结果:
* 有缺货的订单项: 警示页面
* 正常结果: 标准页面, 每一项订单包括多项属性(时间、名称、数量、价格), 分页显示
* 未知的客户ID: 错误页面
